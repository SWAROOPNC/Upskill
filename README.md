# Upskill

This repository contains my code, notebooks, and supporting files from courses in my personal upskill plan

## Purpose

I use this repo to:

- keep all course work in one place
- track progress across different learning paths
- store practice projects, experiments, and notes
- build a reusable reference library from what I learn

## Repository Structure

Course material is organized by program, course, and module when needed.

Current structure:

```text
Upskill/
├── IBM RAG and Agentic AI Professional Certificate/
│   └── 05 Build Multimodal Generative AI Applications/
│       └── M1 Foundations Of Multimodal AI/
│           ├── Use Mixtral and gTTS to create your personal storyteller.ipynb
│           └── generated_story.mp3
└── README.md
```

## What You’ll Find Here

- Jupyter notebooks from hands-on lessons
- generated assets such as audio, images, or outputs
- small practice projects and experiments
- course-specific code files and references

## Naming Convention

To keep things easy to navigate, I generally follow this pattern:

```text
<Program or Certificate>/<Course>/<Module>/
```

Example:

```text
IBM RAG and Agentic AI Professional Certificate/
  05 Build Multimodal Generative AI Applications/
    M1 Foundations Of Multimodal AI/
```

## How I Use This Repo

When adding new learning material, I try to:

1. create a folder for the certificate, course, and module
2. keep notebooks, scripts, and outputs together
3. use clear file names based on the lesson or exercise
4. commit progress regularly
